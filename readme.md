# GATEWAYS-2025-National Level Fest Data Analysis

This project is a Streamlit dashboard built to analyze the data and feedback from the GATEWAYS-2025 National Level Fest. It visualizes participant trends, feedback text analysis using WordClouds, mapping data with GeoPandas, and includes an interactive dashboard with dynamic filtering.

## Features
- **Participants Trends**: View overall participant statistics including total participants, colleges, events, and a Statewise participation map of India.
- **Participants Text Feedback**: See raw and cleaned participant feedback along with an interactive WordCloud highlighting the most frequent words.
- **Participants Ratings**: View min, max, average ratings, and a rating distribution bar chart.
- **Interactive Dashboard**: Filter data dynamically by Event Type and State. Includes breakdowns for Event Types and Average Rating by Event Type.
